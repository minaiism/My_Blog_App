# Simple Blog Application created by using RESTful(backend) and Semantic UI(as a frontend framework)

## Features:

1. Creating new blog
2. Editing, updating and deleting an existing one

## Developed with:

- REST
- Semantic UI
- Express
- Mongoose
- EJS
